\begin{abstract}
Real-world simultaneous speech translation (SST) is rarely context-free: lectures and scientific talks are accompanied by slides, glossaries, speaker biographies, abstracts, and background documents.  Yet streaming systems must emit translations under latency constraints, making external context a double-edged sword: it can disambiguate technical terms and homophones, but it can also induce premature or incorrect commitments when slides are asynchronous, noisy, or only loosely related to the current speech.  We introduce \taskname, a slide-aware contextual SST setting where the model receives a speech stream together with slides, glossary entries, and optional background documents.  We construct \datasetname, a Chinese-to-English lecture challenge set targeting homophones, technical terms, named entities, and slide-speech mismatches.  We further propose \policyname, a lightweight evidence-aware context manager that retrieves small evidence packets and decides whether, when, and how strongly to condition the translator on each source.
